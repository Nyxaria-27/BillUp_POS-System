# Screenshots Folder

Folder ini berisi screenshot aplikasi BillUp untuk dokumentasi README.

## Screenshot yang Dibutuhkan:

1. **welcome.png** - Halaman landing/welcome page
2. **login.png** - Halaman login dengan gradient blue background
3. **register.png** - Halaman register dengan gradient green background
4. **admin-dashboard.png** - Dashboard admin dengan statistik (total penjualan, transaksi, produk)
5. **products.png** - Halaman manajemen produk (list produk dengan gambar)
6. **categories.png** - Halaman manajemen kategori
7. **pos.png** - Interface Point of Sales (cashier) dengan produk dan keranjang
8. **invoice.png** - Contoh invoice/struk pembayaran
9. **transaction-history.png** - Halaman riwayat transaksi
10. **reports.png** - Halaman laporan keuangan (jika ada)

## Tips untuk Screenshot:

- Gunakan resolusi minimal 1920x1080 untuk kualitas terbaik
- Capture dalam mode full-width browser
- Pastikan ada data dummy yang terlihat bagus
- Untuk mobile view, gunakan DevTools responsive mode
- Format file: PNG untuk kualitas terbaik

## How to Take Screenshots:

1. Jalankan aplikasi (`php artisan serve`)
2. Login sebagai Admin atau Cashier
3. Navigate ke halaman yang ingin di-screenshot
4. Tekan `F12` untuk DevTools, pilih responsive mode untuk mobile view
5. Gunakan `Win + Shift + S` (Windows) atau ekstensi browser untuk capture
6. Save dengan nama yang sesuai di folder ini
